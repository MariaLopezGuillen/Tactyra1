# Tactyra ⚽

**Tactyra** es una plataforma web de análisis táctico para **fútbol base**, diseñada para entrenadores y jugadoras. Permite registrar sesiones, analizar acciones ofensivas y defensivas, y generar informes automáticos con métricas y recomendaciones.  

El proyecto utiliza **Laravel**, **MySQL**, **Java (Spring Boot)** y tecnologías web modernas (**HTML, CSS, JS**) para crear una solución escalable y profesional.

---

## 🎯 Objetivos del proyecto

- Analizar partidos y entrenamientos de fútbol base.  
- Evaluar acciones ofensivas y defensivas por jugadora.  
- Generar informes automáticos y métricas avanzadas.  
- Ofrecer un sistema interactivo con Canvas para dibujar jugadas.  

---

## 🧩 Tecnologías utilizadas

- **Backend:** Laravel 10  
- **Microservicio de análisis:** Java Spring Boot  
- **Base de datos:** MySQL  
- **Frontend:** HTML, CSS, JS (Canvas + Charts)  
- **Extras:** Chart.js para visualización de métricas  

---

## 🚀 Funcionalidades principales

1. **Roles y autenticación:** Entrenador / Jugadora  
2. **Gestión de equipos y sesiones**  
3. **Registro de acciones tácticas** (pases, presiones, repliegues…)  
4. **Visualización de jugadas en Canvas**  
5. **Cálculo de métricas avanzadas con microservicio Java**  
6. **Generación de informes PDF automáticos**  
7. **Gráficos de evolución y comparativas**
